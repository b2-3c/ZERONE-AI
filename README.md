## ✨ المميزات الرئيسية

### 💬 المحادثة
- دعم **محادثات متعددة** مع حفظ السجل الكامل
- **تعديل وحذف** الرسائل في أي وقت
- **إعادة توليد** ردود الذكاء الاصطناعي
- **تشعّب المحادثات** — إنشاء فرع من أي رسالة
- **مشاركة الملفات والصور** مباشرة في المحادثة
- **نمط النافذة الصغيرة** للأسئلة السريعة
- **بحث داخل المحادثات**

### 🎨 الواجهة
- تصميم حديث باستخدام **GTK4 و Adwaita**
- **6 ثيمات ألوان** قابلة للتخصيص
- دعم **الوضع الداكن والفاتح**
- واجهة عربية كاملة مع دعم متعدد اللغات

### 📚 المكتبة
- حفظ **القوالب والشخصيات والملفات والعناصر** المفضلة
- استخدام العناصر المحفوظة في أي محادثة بنقرة واحدة
- **بحث وتصفية** داخل المكتبة

### 🔊 تحويل النص إلى صوت
- صفحة مخصصة لتحويل أي نص إلى صوت
- اختيار الصوت واللغة مباشرة
- **تحميل الصوت** كملف WAV
- تحكم كامل في **سرعة التشغيل**

### 🧠 الذاكرة طويلة الأمد
- حفظ معلومات المستخدم عبر المحادثات المختلفة
- محركات ذاكرة متعددة تناسب الاحتياجات المختلفة
